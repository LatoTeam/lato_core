    core_getCacheDirectory() click to toggle source

    Funzione che ritorna il percorso alla directory di cache usata dai moduli lato per memorizzare informazioni sull'applicazione principale. La directory dell'app principale usata come cache si trova nel percorso #{Rails.root}/config/lato

    # File lib/lato_core/interface/cache.rb, line 10
def core_getCacheDirectory
  dirname = "#{Rails.root}/config/lato"
  FileUtils.mkdir_p(dirname) unless File.exist?(dirname)
  dirname
end
